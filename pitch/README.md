# Pitch Detection

## Description
This Python program takes in a .wav file, reads the file, and outputs the primary pitch per sample

## Requirements
* Python3
#### Python Modules
* sys
* math
* aubio `pip3 install aubio`
* pyaudio
* numpy

## Run
### wav file analysis
`python3 pitch.py filename.wav`
### real time analysis
`python3 pitch.py`
